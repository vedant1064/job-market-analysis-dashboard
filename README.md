# 📊 Job Market Analysis Dashboard (Power BI)

## 🔍 Overview

This project analyzes job market trends using Power BI. It provides insights into job distribution, salary trends, hiring companies, and geographic demand.

## 📁 Dataset

Includes:

* Job Title
* Salary Estimate
* Company Name
* Industry
* Location
* Company Size

## ⚙️ Tools Used

* Power BI
* Power Query
* Data Modeling

## 📈 Key Insights

* Biotech & Insurance have highest job demand
* Finance sector dominates
* Average salary ≈ 102K
* Large companies hire more
* Jobs concentrated in major cities
